---background-position: x-position y-position;
background-position: keyword;

| Value           | Description                             |
| --------------- | --------------------------------------- |
| `left top`      | Top-left corner (default)               |
| `left center`   | Left side, vertically centered          |
| `left bottom`   | Bottom-left corner                      |
| `right top`     | Top-right corner                        |
| `right center`  | Right side, vertically centered         |
| `right bottom`  | Bottom-right corner                     |
| `center top`    | Center horizontally, top vertically     |
| `center center` | Center both horizontally and vertically |
| `center bottom` | Center horizontally, bottom vertically  |

--background-position: center center;
background-position: 50% 50%; /* Center */
background-position: 0 0; /* Top-left */
background-position: 100% 100%; /* Bottom-right */
background-position: 10px 20px; /* 10px from left, 20px from top */

--background-position: right; /* right center */
background-position: 20px; /* 20px from the left, vertically centered */

--body {
  background-image: url("image.jpg");
  background-repeat: no-repeat;
  background-position: right bottom;
  background-size: cover;
}
